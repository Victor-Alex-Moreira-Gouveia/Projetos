Documentação sobre docker

Comandos CMD docker:


  • docker ps -a: Lista container em execução.
  • docker rm "IDcontainer": Remove container escolhido.
  • docker image ls: Lista as imagens instaladas.
  • docker rmi "IDimage": Remove a imagem escolhido.
  • docker build -t "Nome_da_imagem" .: constroi uma imagem docker apartir de um dockerfile
  • docker run -it --name "Nome_container" -p 80:8080 "Name_image": Cria um container partir de uma imagem docker no modo interativo
